```bash
# 1. Generate fresh metrics
bash tools/inventory.sh > tools/outputs/inventory.json
bash tools/dep-graph.sh > tools/outputs/dep-graph.txt

# 2. Review audit report
open docs/architecture-audit/audit-report.md

# 3. Check refactor backlog
open docs/architecture-audit/refactor-tickets.md
```
