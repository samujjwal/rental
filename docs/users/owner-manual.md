# Owner Manual

This document summarizes the main owner-facing journeys supported by the platform.

## Core Tasks

- create and edit listings
- upload media and fill category-specific listing fields
- manage pricing and availability
- review and respond to bookings
- message renters
- monitor payouts, insights, and organization-related surfaces
- manage insurance-related workflows where enabled

## Typical Journey

1. Sign in as an owner.
2. Create or update a listing.
3. Configure pricing, availability, and listing details.
4. Review booking activity and take allowed lifecycle actions.
5. Track operational status through dashboard and settings surfaces.

## Channels

- web supports the broadest owner workflow coverage
- mobile supports core owner actions, with narrower parity than web admin and ops surfaces
