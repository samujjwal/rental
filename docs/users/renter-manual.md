# Renter Manual

This document summarizes the main renter-facing journeys supported by the platform.

## Core Tasks

- search and browse listings
- filter by category, price, location, and related listing attributes
- review listing detail, availability, pricing, and trust signals
- create a booking and complete checkout when required
- message owners
- manage favorites, notifications, profile, and booking history
- leave reviews or open disputes when eligible

## Typical Journey

1. Search for a listing.
2. Open the listing detail page.
3. Choose dates and review pricing.
4. Create a booking or proceed to checkout.
5. Track booking state and communicate with the owner.
6. Review the experience or resolve issues if needed.

## Channels

- web supports the broadest renter feature surface
- mobile supports core renter flows including search, booking-related actions, messaging, and settings
